# Edge & Edge — A Perimeter Company

> The next-generation blockchain perimeter platform. We operate at the edge. Of what, we are not at liberty to disclose.

A single-page satirical landing site. One self-contained `index.html` — embedded CSS, embedded JS, zero build step, zero dependencies. Operating at the edge since last Tuesday.

## Run it

It's a static file. Any of these will do:

```sh
# Built-in dev server (Python)
npm run dev

# ...or just open it
open index.html
```

Then visit http://localhost:4321.

## Deploy

Pure static — no config required. From this directory:

```sh
vercel        # preview
vercel --prod # production
```
